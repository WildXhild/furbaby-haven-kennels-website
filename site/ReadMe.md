# Furbaby Kennels Website

## Overview
Static business website built from scratch using HTML, CSS, and JavaScript in VS Code.  
Deployed live on Netlify to showcase available puppies and facilitate customer inquiries.  
Prompt engineering was used to assist in design and layout ideas.

## Live Site
https://furbabykennels.site/

## Features
- Responsive layout for mobile and desktop
- Puppy listings and images
- Contact and inquiry information
- Simple, user-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Deployed via Netlify

## What I Learned
- Structuring a real-world static website
- Writing maintainable HTML, CSS, and JavaScript
- Deploying and managing a live site
- Using prompt-assisted design for UI/UX ideas
